# 使用说明

本项目为 Python 项目，建议使用虚拟环境进行开发和运行。

## 安装依赖

1. 创建虚拟环境（可选）：
   ```
   python3 -m venv venv
   ```
   或者是
   ```
   python -m venv venv
   ```
2. 激活虚拟环境：
   - Windows:
     ```
     venv\Scripts\activate
     ```
   - macOS/Linux:
     ```
     source venv/bin/activate
     ```
3. 安装依赖（如有 requirements.txt 文件）：
   ```
   pip3 install -r requirements.txt
   ```
   或者
   ```
   pip install -r requirements.txt
   ```

## 构建文档

1. 使用 jupyter-book 构建文档：
   ```
   jupyter-book build .
   ```

2. 更多常用命令示例：

   - 预览生成的书籍（默认输出在 _build/html 文件夹）：
     ```
     open _build/html/index.html
     ```
     或在 Windows 下：
     ```
     start _build/html/index.html
     ```

   - 清理构建生成的文件：
     ```
     jupyter-book clean .
     ```

   - 检查文档配置和内容是否有问题：
     ```
     jupyter-book config sphinx .
     ```

   - 生成 PDF（需额外依赖）：
     ```
     jupyter-book build . --builder pdflatex
     ```
