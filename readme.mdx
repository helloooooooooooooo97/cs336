# 使用说明

本项目为 Python 项目，建议使用虚拟环境进行开发和运行。

## 安装依赖

1. 创建虚拟环境（可选）：
   ```
   python3 -m venv venv
   ```
   或者是
   ```
   python -m venv venv
   ```
2. 激活虚拟环境：
   - Windows:
     ```
     venv\Scripts\activate
     ```
   - macOS/Linux:
     ```
     source venv/bin/activate
     ```
3. 安装依赖（如有 requirements.txt 文件）：
   ```
   pip3 install -r requirements.txt
   ```
   或者
   ```
   pip install -r requirements.txt
   ```

## 构建文档

1. 使用 jupyter-book 构建文档：
   ```
   jupyter-book build .
   ```

2. 更多常用命令示例：

   - 预览生成的书籍（默认输出在 _build/html 文件夹）：
     ```
     open _build/html/index.html
     ```
     或在 Windows 下：
     ```
     start _build/html/index.html
     ```

   - 清理构建生成的文件：
     ```
     jupyter-book clean .
     ```

   - 检查文档配置和内容是否有问题：
     ```
     jupyter-book config sphinx .
     ```

   - 生成 PDF（需额外依赖）：
     ```
     jupyter-book build . --builder pdflatex
     ```

## makefile命令
也可以使用下面这些makefile命令快速构建
常用 makefile 命令如下：

- 构建虚拟环境（只需执行一次）：
  ```
  make venv
  ```

- 安装依赖（需先构建虚拟环境）：
  ```
  make install
  ```

- 激活虚拟环境（需手动执行，命令会提示如何激活）：
  ```
  make activate
  ```

- 构建 jupyter-book 文档：
  ```
  make build
  ```

- 打开生成的 HTML 首页（根据操作系统自动判断）：
  ```
  make open
  ```

> 提示：如需查看所有可用命令，可运行 `make` 或查看 makefile 文件内容。
